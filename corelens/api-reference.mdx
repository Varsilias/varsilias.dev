This page is a compact reference for the public Corelens API shown throughout the docs.

## Imports

```ts
import { corelens, PrometheusTextExporter } from '@varsilias/corelens';
import { ExpressMetricsAdapter } from '@varsilias/corelens/adapter';
import { CorelensModule } from '@varsilias/corelens/adapter/nest';
import type { CorelensConfig } from '@varsilias/corelens/core';
```

Published entry points:

- `@varsilias/corelens`
- `@varsilias/corelens/adapter`
- `@varsilias/corelens/adapter/nest`
- `@varsilias/corelens/core`
- `@varsilias/corelens/exporters`
- `@varsilias/corelens/modules`

Express, Fastify, and Hono adapters are exported from `@varsilias/corelens/adapter`. NestJS exports are isolated under `@varsilias/corelens/adapter/nest` because the Nest adapter depends on Nest runtime packages such as `@nestjs/common`.

## `corelens(config)`

Creates a Corelens instance.

```ts
const lens = corelens({
  serviceName: 'orders-api',
  logs: { enabled: true },
  metrics: { enabled: true },
  traces: { enabled: true, samplingRate: 0.25 },
  export: { enabled: false },
});
```

## Logger

```ts
lens.logger.info('payment accepted', {
  orderId: 'ord_123',
  amount: 42_00,
});

lens.logger.warn('inventory low', { sku: 'sku_123' });
lens.logger.error('payment failed', { orderId: 'ord_123', reason: 'timeout' });
```

## Metrics

```ts
const ordersTotal = lens.metrics.counter(
  'orders_total',
  'Total orders created',
);

ordersTotal.inc({ status: 'created' });
```

```ts
const orderValue = lens.metrics.histogram(
  'order_value',
  'Order value in major currency units',
  { buckets: [10, 25, 50, 100, 250, 500] },
);

orderValue.observe(42, { currency: 'USD' });
```

## Tracing

```ts
await lens.tracer.withSpan('orders.reserveInventory', async (span) => {
  span?.setAttribute('component', 'inventory');
  span?.addEvent('inventory.reserve.started');
});
```

```ts
const response = await lens.tracer.withClientSpan(
  {
    name: 'GET catalog',
    attributes: {
      'http.method': 'GET',
      'http.url': 'https://catalog.internal/recommendations',
    },
  },
  async ({ traceparent, span }) => {
    const response = await fetch('https://catalog.internal/recommendations', {
      headers: { traceparent },
    });

    span?.setAttribute('http.status_code', response.status);
    return response;
  },
);
```

## Prometheus exporter

```ts
const prometheus = new PrometheusTextExporter();
const body = prometheus.render(lens.metrics.snapshot());
```

## NestJS adapter

```ts
import {
  CorelensModule,
  CorelensService,
  CorelensNestLogger,
  CorelensHttpInterceptor,
} from '@varsilias/corelens/adapter/nest';
```

`CorelensModule.forRoot()` registers Corelens in Nest dependency injection. `CorelensModule.forRootAsync()` supports factory-based async config through `imports`, `inject`, and `useFactory`.

`CorelensService` exposes the Corelens logger, metrics registry, tracer, HTTP recorders, and `snapshot` stats accessor to injectable Nest providers. `CorelensNestLogger` implements Nest's `LoggerService`. `CorelensHttpInterceptor` records Nest HTTP metrics and tracing with parameterized route labels.

See [NestJS adapter](/corelens/framework-adapters/nestjs) for setup, logger wiring, interceptor registration, and Prometheus endpoint guidance.

## Stats

```ts
const stats = lens.getStats();
```

Use stats to inspect exporter failures, queue state, dropped items, shutdown result, and trace processor state.

## Shutdown

```ts
const result = await lens.shutdown();
```

Call shutdown before process exit when export is enabled.
