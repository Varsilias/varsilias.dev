Use symptoms first. Then check configuration, instrumentation placement, and exporter delivery.

## No telemetry appears

Check these items:

1. Confirm `createCorelens()` runs before request handling starts.
2. Confirm middleware is registered before route handlers.
3. Use the console exporter to remove collector delivery from the problem.
4. Confirm sampling is not set to `0`.

## Spans are missing request context

The explicit span may be running outside the request context. Keep async work inside the request path or pass context explicitly when work moves to a queue.

## Metrics have too many labels

High-cardinality labels can make dashboards noisy and expensive. Remove IDs, emails, tokens, and unbounded user input from metric labels.

## OTLP export fails

Check the collector endpoint and network path from the service container.

```bash
curl -v http://otel-collector:4318
```

If the collector is unavailable, Corelens should not fail customer requests. Fix exporter delivery separately from request handling.
