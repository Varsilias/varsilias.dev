Use the local workflow to catch broken links and formatting issues before review.

## Edit docs

Update MDX pages in the relevant namespace. Add shared content to `snippets/` only when the exact text is reused.

## Preview locally

```bash
mint dev
```

## Check links

```bash
mint broken-links
```

## Validate the site

```bash
mint validate
```
