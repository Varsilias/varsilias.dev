Varsilias.dev is the documentation home for the open source tools in this ecosystem. The project exists so contributors can improve the docs, examples, and technical explanations around each tool.

## What belongs here

- Product documentation for tools such as Corelens and Concile
- Architecture notes that explain tradeoffs
- Quickstarts and troubleshooting guides
- Contribution workflows for docs changes
- Shared writing and layout conventions

## What does not belong here

- Private operational notes
- Undocumented product claims
- Marketing copy without technical value
- API behavior that has not been verified

## How the site is organized

Each tool owns its own namespace and sidebar group. Varsilias.dev owns the contribution and documentation-system guides.

```text
corelens/
concile/
varsilias-dev/
snippets/
docs.json
style.css
```
